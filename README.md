# Pure ECS RTS with Unity3D
Trying to recreate a simple RTS game using Unity and pure ECS

# Core APIs used:
- IComponentData
- JobComponentSystem
- IJobProcessComponentData<T>
- BarrierSystem
- EntityCommandBuffer
- ComponentDataFromEntity
